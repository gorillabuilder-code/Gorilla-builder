Quickstart: Your First App

Let's take your idea from a single sentence to a live, full-stack application in under 3 minutes. This guide will walk you through the entire autonomous lifecycle of a Gor://a Builder project.

Step 1: Crafting the Initial Prompt

From the Dashboard, locate the main prompt bar. You have three powerful ways to initiate a build:

  1. Text Prompt: Type your idea explicitly. Be descriptive about functionality, backend requirements, and layout.
  2. Image Reference: Click the icon to upload a screenshot of a UI you want the AI to replicate pixel-for-pixel.
  3. Figma Link: Paste a Figma URL to trigger our native Figma Compiler. Our AgentSkills will parse the design nodes and seamlessly convert them into a Tailwind TSX layout.
// Example of a great initial prompt:
"Build a SaaS dashboard for tracking SEO metrics. 
Include a Node.js backend with Supabase integration to store user data. 
Add Google Sign-in, and ensure the UI uses dark mode glassmorphism."

Step 2: Watch the Swarm Choreography

Once you hit submit, you'll enter the Editor. On the left side, the Agent Activity feed will show the Multi-Agent Swarm thinking out loud via our internal MCP (Machine Communication Protocol):

  • Phase 1: Planning. The Planner maps out the React components and the Express API routes.
  • Phase 2: Coding. The Coder reads your existing boilerplate files, generates the new Node.js infrastructure, and wires the frontend React hooks to your backend APIs.
  • Phase 3: Saving. The AI commits the 22,000+ lines of infrastructure to the Virtual File System.

Do Not Interrupt the Swarm: Gor://a prioritizes high-quality, full-file rewrites over patchy snippets. A complex full-stack prompt may take 60 to 90 seconds to compile. Sit back and watch the architecture unfold.

Step 3: The WebContainer Boot Sequence

Once the AI commits the code, a live, headless Node.js environment boots up directly inside your browser tab.

  1. Dependency Sync: The system automatically runs pnpm install to fetch any libraries the AI utilized.
  2. Dev Server Start: It runs npm run dev, booting both the Vite frontend and the Express backend simultaneously.
  3. The Gateway Injection: If you requested AI features or Auth, the Gor://a Gateway automatically injects the necessary security headers and routes. Zero API keys required.

Step 4: Auto-Healing & Iteration

Unlike other platforms that leave you stranded with a broken app, Gor://a features a Frenzy Vacuum and an Auto-Healing Loop. If your app throws a Node.js exception or a React syntax error, the WebContainer intercepts the stack trace, waits for the error cascade to finish, and silently passes the logs back to the AI Debugger to fix it autonomously.

Step 5: One-Click Vercel Deployment

When you are ready to launch your SaaS to the world, click the Deploy button. Gor://a will automatically optimize your server.js for serverless edge functions, push the code to a private GitHub repository, and trigger a live Vercel deployment. You just went from prompt to production in one sitting.
