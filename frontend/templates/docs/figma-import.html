How to Import from Figma

A step-by-step guide to linking your account and extracting your first design node.

Step 1: Link Your Figma Account

Before you can extract designs, Gor://a Builder needs permission to read your Figma files. We use secure OAuth to do this.

  1. Navigate to Account & Settings from the dashboard sidebar.
  2. Find the Figma integration block and click Link Account.
  3. You will be redirected to Figma. Grant Gor://a the file_content:read permission.
  4. You will be bounced back to the Dashboard. The Figma icon in the prompt bar will now be active.

Step 2: Get the Correct Figma URL

This is the most critical step. Gor://a's compiler needs to know exactly which component or screen to build. Do not paste the generic link to your entire Figma file.

  1. Open your project in Figma.
  2. Click on the specific Frame or Component you want to build. (Ensure you have selected the top-level parent frame, not a single text layer inside it).
  3. Right-click the frame, navigate to Copy/Paste as, and select Copy link.
  4. The URL should look something like this: https://www.figma.com/design/abc123XYZ/App?node-id=1-234&t=...

The node-id is Mandatory: If your URL does not contain ?node-id=..., the Gor://a backend will reject it. The AST compiler cannot compress an entire multi-page canvas; it requires a targeted node.

Step 3: Trigger the Build

Go back to the Gor://a Dashboard. Click the Figma icon in the prompt bar. The input field will turn green.

Paste your copied Figma URL and hit Enter. Gor://a will intercept the URL, run the Smart Fidelity Compression, deduct the specific tokens used for the compilation, and inject the finished React code directly into your WebContainer environment before the Swarm even boots.

Troubleshooting Imports

  • Error: "Could not find that specific node": Ensure the frame you linked still exists and hasn't been deleted or moved to a different file.
  • Missing Icons: If you use complex vector math (pen tool drawings) instead of standard icon fonts, Gor://a will replace them with a <lucide-icon> placeholder to keep the code clean. You can ask the AI to swap these out later.
  • Empty Frames: If you link to a frame that has no background color, no borders, and no text, the compressor will purge it to save tokens, resulting in a blank output. Ensure your frame has actual UI elements.
