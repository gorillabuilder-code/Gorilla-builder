Troubleshooting

Stuck? Here are solutions to the most common issues.

1. "Server Stopped" or Red Screen

If you see a red "Connection Refused" screen in the preview:

  • Wait 5 seconds: The container might just be waking up.
  • Click Refresh: Hit the refresh icon in the preview header.
  • Check the Chat: Did the AI report a crash? If so, it is likely already fixing it.

2. "404 Not Found"

If your app loads but says "404":

  • For Standard Projects: Ensure you have an `index.html` file.
  • For X-Mode: Ensure your `vite.config.ts` has the correct base path. The Auto-Fixer usually handles this.

3. The AI is Loop-Fixing

Sometimes the AI gets stuck trying to fix the same error over and over. If this happens:

  1. Stop the server (Refresh the page).
  2. Give the AI a New Prompt with a hint (e.g., "Stop trying to use that library, use standard CSS instead").

4. Usage Limits

If the chat stops responding, you may have hit your monthly token limit. Check your Dashboard widget.
