Understanding API Tokens

The currency of the Gor://a ecosystem. How we fuel the Swarm without bankrupting you.

The VybeCoding Paywall Trap

The standard AI development platform uses a predatory pricing model. They offer a "free tier" that lasts exactly long enough to build a basic HTML page. The moment you ask for a backend or a complex component, you hit a hard paywall demanding $30 to $40 a month. They treat infrastructure like a luxury.

Gor://a Builder treats infrastructure as a right. Because we run your compute locally via WebContainers and monetize the free tier through non-intrusive ads, we pass the server savings directly to you in the form of Tokens.

Your Starting Balance: 500,000 Credits

Every new user on the Free Tier starts with a massive 500,000 Free Tokens every month. This is enough to plan, build, and deploy several full-stack applications. If you upgrade to Premium, your limit expands to a staggering 5,000,000 Tokens.

How Tokens are Calculated

Tokens are burned whenever the Gor://a Swarm performs cognitive or computational labor. Our Unified AI Gateway tracks this usage invisibly:

  • Code Generation (The Swarm): When the Planner, Reasoner, or Coder agents process your prompt and write React/Node.js code, tokens are consumed based on the context length. Tip: Use the "Expert" Code Style in Agent Skills to generate more concise code and save tokens.
  • In-App LLM Chat: If your generated app uses the /api/v1/chat/completions endpoint, we charge a highly optimized fraction (usually 0.3 to 0.5 credits per raw API token) to route it through our high-performance models.
  • Image Generation (Fireworks AI): Generating an image via the Gateway costs a fixed, flat rate of 250 tokens.
  • Speech-to-Text (Whisper): Audio transcription costs exactly 100 tokens per estimated minute of audio.

What is 100% Free? Background removal via RemBG costs 0 tokens. Text-to-Speech (TTS) using the browser's native synthesis costs 0 tokens. Running the WebContainer and hosting your local server costs 0 tokens.

Context Length Management

To prevent the AI from accidentally blowing through your entire token balance by reading a massive, 10,000-line file, the Gor://a backend uses an intelligent Context Manager. It automatically shortens conversational history, purges irrelevant logs, and utilizes a "Frenzy Vacuum" to deduplicate error reports before sending them to the LLM.
