How the AI Agent Works

Gorilla is not just a text generator; it is an autonomous agent that plans, codes, and fixes itself. To build apps with AI features

The 3-Step Process

Phase 1: Planning

When you send a request, the Planner Agent analyzes your file structure. It creates a step-by-step "Todo List" of what files need to be created or modified.

Phase 2: Coding

The Coder Agent picks up the plan. It writes the code for each file.

  • It knows about modern frameworks (React, Vue, Tailwind).
  • It automatically manages imports and dependencies.

Phase 3: Healing (Auto-Fix)

This is the magic part. After writing the code, Gorilla tries to start the server. If the server crashes (e.g., a missing comma or wrong import), Gorilla reads the error log, understands what went wrong, and automatically writes a patch to fix it.

You might see messages like "Analyzing Crash..." or "Auto-Fixing". This is normal! It means the agent is polishing the code for you.

Best Practices

  • One thing at a time: Don't ask for a database, a login system, and a payment gateway in one message. Break it down.
  • Be patient: Complex changes might take 30-60 seconds as the AI thinks and writes.
