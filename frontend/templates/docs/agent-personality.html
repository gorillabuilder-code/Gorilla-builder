Agent Personality

Customize the tone and vibe of the AI's chat interface in the Editor.

It's Your Workspace, Your Vibe

When you are in the Gor://a Editor, the Agent Activity feed and the chat window are your primary interfaces for interacting with the Swarm. The Agent Personality setting dictates how the Planner, Reasoner, and Coder agents format their conversational updates to you.

Code Quality Remains Unchanged: This setting only affects the conversational text (the assistant_message JSON field). It does not affect the actual quality, syntax, or functionality of the generated React and Node.js code.

Option A: Professional

For a strictly business, distraction-free environment. The agents will strip out enthusiasm, emojis, and filler words, giving you clinical, direct updates on system state and file operations.

  • Typical Planner Response: "I have analyzed the request. Generating the necessary Express routes and React components for the authentication module."
  • Typical Coder Response: "Implementation complete. Updating server.js and userProfile.tsx."

Option B: Casual & Friendly

For the classic Indie Hacker experience. The agents will act as enthusiastic co-founders, using emojis, conversational language, and celebrating successful builds.

  • Typical Planner Response: "Awesome idea! 🚀 I've put together a plan to build out the auth module. Let's get to work!"
  • Typical Coder Response: "All done! 🎉 I've wired up the backend routes and the UI looks super sleek. Check out the preview!"

Why Personality Matters

Beyond just preference, the Agent Personality affects the Agent Thought Timeline in the editor. A more concise, professional personality will generate shorter, scannable logs, allowing you to debug complex architectural decisions faster. A casual personality makes the platform feel more approachable and engaging during long, late-night coding sessions.
