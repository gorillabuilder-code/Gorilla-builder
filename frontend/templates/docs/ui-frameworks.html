UI Frameworks

Lock the Coder Agent into your preferred CSS architecture.

Why Global CSS Enforcement Matters

One of the most common ways AI coding assistants break applications is by mixing CSS paradigms. In one prompt, the AI might use Tailwind classes; in the next, it hallucinates a massive styles.css file; in the third, it resorts to inline style={{}} tags. This results in an unmaintainable codebase.

Gor://a Builder solves this by strictly enforcing your chosen UI Framework at the system-prompt level.

Option A: Tailwind CSS (Recommended)

When you select Tailwind CSS, the Coder Agent is locked into using utility classes directly in your React JSX/TSX files. It will explicitly avoid creating external CSS files unless absolutely necessary for complex keyframe animations.

Gor://a’s Golden Boilerplate is heavily optimized for Tailwind, meaning the WebContainer environment already has tailwind.config.js and PostCSS configured and running the moment your project boots.

// Example Output: Tailwind CSS
export default function Card({ title, content }) {
  return (
    <div className="p-6 bg-slate-900 border border-slate-800 rounded-2xl shadow-xl hover:shadow-2xl transition-all">
      <h3 className="text-xl font-bold text-white mb-2">{title}</h3>
      <p className="text-slate-400 text-sm leading-relaxed">{content}</p>
    </div>
  );
}

Option B: Vanilla CSS

For CSS purists, or for developers building highly specific, custom-designed landing pages, you can force the agent to use Vanilla CSS. The Coder agent will structure its operations to modify src/index.css (or generate scoped CSS modules) and use standard semantic class names in your components.

// Example Output: Vanilla CSS
/* In src/index.css */
.premium-card {
  padding: 24px;
  background-color: #0f172a;
  border: 1px solid #1e293b;
  border-radius: 16px;
}
.premium-card-title {
  font-size: 20px;
  color: #ffffff;
}

// In your React Component
export default function Card({ title, content }) {
  return (
    <div className="premium-card">
      <h3 className="premium-card-title">{title}</h3>
    </div>
  );
}

The Shadcn/UI Synergy: If you select Tailwind CSS, the Gor://a agent is also given permission to utilize the pre-installed Shadcn/UI component library located in src/components/ui/. This allows the AI to instantly drop in accessible, highly-polished dropdowns, dialogs, and form elements without writing them from scratch.
