Code Style

Optimize for readability and learning, or optimize for speed and token economics.

The Balance of Tokens vs. Readability

Every character generated by the Coder Agent costs tokens. Comments, long variable names, and verbose explanations consume your monthly quota. Gor://a Builder allows you to choose whether you want the AI to act as a patient teacher, or a ruthless, highly-efficient senior engineer.

Option A: Beginner Friendly (Heavily Commented)

If you are learning React, Node.js, or how Supabase connects to an Express backend, you should select the Beginner style.

The Coder Agent will aggressively comment its code, explaining why a React useEffect hook is structured a certain way, or how a database query works. It will use highly descriptive variable names and avoid overly complex shorthand.

// Example Output: Beginner Friendly
export default function UserProfile({ userId }) {
  // 1. Create a state variable to hold the user's data. 
  // We initialize it as 'null' while loading.
  const [userData, setUserData] = useState(null);

  // 2. Use the useEffect hook to fetch data when the component mounts
  useEffect(() => {
    async function fetchUserFromDatabase() {
      try {
        // We call our Express backend route here
        const response = await fetch(`/api/users/${userId}`);
        const data = await response.json();
        setUserData(data);
      } catch (error) {
        console.error("Failed to load the user profile:", error);
      }
    }
    
    // Execute the fetch function
    fetchUserFromDatabase();
  }, [userId]); // Re-run if userId changes
  
  // ... rest of component
}

Option B: Expert / Concise (The Token Saver)

If you are an experienced developer who just wants the app built as fast as possible, select Expert. The Coder Agent will strip out all unnecessary comments, utilize advanced DRY (Don't Repeat Yourself) principles, and write highly optimized, concise TypeScript and ES6+ code.

The Token Hack: Because the Expert mode strips out conversational comments and compresses the logic, it generates significantly fewer tokens. If you are on the Free Tier managing a 500,000 token limit, switching to Expert Mode is the easiest way to stretch your credits further.

// Example Output: Expert / Concise
export default function UserProfile({ userId }) {
  const [user, setUser] = useState(null);

  useEffect(() => {
    fetch(`/api/users/${userId}`)
      .then(res => res.json())
      .then(setUser)
      .catch(console.error);
  }, [userId]);
  
  // ... rest of component
}
