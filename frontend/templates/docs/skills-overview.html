What are Agent Skills?

Stop wrestling with generic AI outputs. Train your Swarm to write code exactly the way you do.

The Problem with "VybeCoding" Defaults

If you've used standard AI coding assistants, you know the frustration of the "default AI voice." You ask for a button, and it gives you a bloated, inline-styled monstrosity. You ask for a clean layout, and it hallucinates six different CSS files. You are forced to waste your time—and your API tokens—typing "make it look modern, use Tailwind, don't use emojis, add comments" into every single prompt.

This is the VybeCoding trap: standard AI tools have amnesia. They don't know who you are, and they don't know your stack.

Enter Agent Skills

Gor://a Builder solves this with Agent Skills. Agent Skills are a global configuration engine that acts as the permanent memory and style guide for your AI Swarm.

When you configure your Agent Skills in the Dashboard, Gor://a compiles your preferences into a strict set of architectural directives. These directives are seamlessly injected into the SYSTEM_PROMPT of the Planner and Coder agents before every single generation.

The Token Advantage: By defining your stack and style globally, you save hundreds of tokens on every prompt. You no longer need to write paragraphs explaining your styling preferences. You just type "Build a pricing page" and the Swarm instantly knows to use Tailwind, Lucide icons, and concise TypeScript.

What Can You Customize?

The Agent Skills wizard allows you to define five core pillars of your development environment:

  1. Visual Preferences: Control how the AI handles iconography (Professional SVGs vs. Emojis).
  2. UI Frameworks: Lock the agent into Tailwind CSS utility classes or Vanilla CSS stylesheets.
  3. Code Style: Choose between heavily-commented beginner code, or ultra-concise expert patterns.
  4. Agent Personality: Dictate how the AI speaks to you in the Editor chat UI.
  5. Golden Rules: Write strict, unbreakable custom rules (e.g., "Never use alert()", "Always build in Dark Mode").

By taking 60 seconds to configure your Agent Skills, you transform Gor://a from a generic code generator into a highly personalized, Senior Full-Stack co-founder.
