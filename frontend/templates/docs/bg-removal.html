Background Removal (RemBG)

Surgical, AI-powered background removal for E-commerce, profile pictures, and design tools.

The "Utility" API

Not every AI integration needs to be a massive LLM. Sometimes, you just need a sharp, reliable utility. Background removal is a staple feature in modern SaaS, particularly for E-commerce product listings, real estate apps, or avatar creators.

Instead of forcing you to sign up for remove.bg or complicated Python microservices, Gor://a Builder runs a high-performance RemBG cluster directly on our edge infrastructure.

The Gateway Endpoint

Your generated Node.js backend can process background removal by forwarding an image file to /api/v1/images/remove-background.

// Example Backend Implementation (routes/tools.js)
import express from 'express';
import multer from 'multer';
import fetch from 'node-fetch';
import FormData from 'form-data';

const router = express.Router();
const upload = multer({ storage: multer.memoryStorage() });

router.post('/remove-bg', upload.single('image'), async (req, res) => {
  const form = new FormData();
  form.append('file', req.file.buffer, req.file.originalname);

  const response = await fetch('https://app.gorillabuilder.dev/v1/images/remove-background', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.GORILLA_API_KEY}`
    },
    body: form
  });

  // The Gateway returns the raw PNG image buffer with a transparent background!
  const imageBuffer = await response.buffer();
  res.set('Content-Type', 'image/png');
  res.send(imageBuffer);
});

export default router;
