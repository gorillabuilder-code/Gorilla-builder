The Editor Workspace

Once you create a project, you land in the Editor. This is a split-screen environment designed for rapid iteration.

1. File Tree (Left)

This shows all the files the AI has generated.

  • Click a file to view its code.
  • Standard web apps usually have `index.html`, `style.css`, and `script.js`.
  • React/Vite apps (X-Mode) will have `src/`, `package.json`, and `vite.config.ts`.

2. Code Editor (Center)

This is a fully functional Monaco editor (like VS Code). You can:

  • Manually edit code if you know how.
  • Copy code to your clipboard.
  • Watch the AI write code in real-time.

3. Live Preview (Right)

This is your actual running app. It is not a screenshot; it is a live web server.

Tip: If the preview is stuck or loading, click the Refresh button in the preview header.

4. Chat Interface (Bottom Right)

This is where you talk to the AI. You can ask for changes, report bugs, or request new features. The AI reads your current code before answering.
