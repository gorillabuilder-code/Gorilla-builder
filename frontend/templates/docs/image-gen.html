Image Generation (Fireworks AI)

Allow your users to generate stunning visual assets inside your app instantly.

Integrating Vision

Whether you are building an AI avatar generator, a marketing asset creator, or a dynamic blog thumbnail tool, generating images programmatically is a highly sought-after SaaS feature. Like text generation, Gor://a handles this through the Unified Gateway.

Behind the scenes, we utilize Fireworks AI (specifically the playground-v2-5-1024px-aesthetic model) to generate incredibly high-fidelity, production-ready images in seconds.

The Gateway Endpoint

When you request image generation capabilities, the Coder Agent routes requests to /api/v1/images/generations. Just like the LLM chat, your Node.js backend uses your GORILLA_API_KEY to authenticate the request.

// Example Backend Implementation (routes/images.js)
router.post('/generate', async (req, res) => {
  const { prompt } = req.body;
  
  const response = await fetch('https://app.gorillabuilder.dev/v1/images/generations', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.GORILLA_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ prompt: prompt })
  });

  const data = await response.json();
  // Returns standard base64 or URL data to be rendered by React
  res.json({ imageUrl: data.base64 }); 
});

Fixed Token Cost: Image generation is computationally heavy. To keep economics predictable, every successful image generated through the Gor://a Gateway costs a flat rate of 250 tokens deducted from your account balance.

How to Prompt for It

To get the best results, instruct the AI on exactly how the frontend should handle the loading state, as image generation can take 3-5 seconds.

Example Prompt: "Create a 'Cyberpunk Avatar Generator'. The user types a description of themselves. The Express backend should send this to the Gor://a image generation gateway. On the React frontend, show a cool glowing loading spinner while waiting, and then display the generated image in a beautifully styled Polaroid frame."
