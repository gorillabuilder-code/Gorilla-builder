Files & Assets

Gorilla gives you full access to the source code of your AI application. You are not locked into a "black box", you own your code.

File Structure

The file tree on the left side of the Editor shows your project structure:

  • Standard Projects: Typically contain `index.html`, `style.css`, and `script.js`. Simple and effective.
  • X-Mode Projects: Follow modern conventions with a `src/` folder, `package.json`, `vite.config.ts`, and component files (e.g., `App.tsx`).

Uploading Images & Assets

Currently, the best way to use images is to link to external URLs (like Unsplash, Imgur, or your own CDN).

Example:

<img src="https://images.unsplash.com/photo-12345..." alt="My Image" />

We are working on a native "Drag & Drop" file uploader which will be released in the next update.

Downloading Your Code

You can export your entire project at any time:

  1. Click the Settings (gear icon) in the top right.
  2. Click Export to ZIP.
  3. This downloads a standard archive you can host anywhere (Netlify, Vercel, or your own server).
